HTML Tags

<html>

Defines the root of an HTML document.

<head>

Contains meta-information about the HTML document.

<title>

Sets the title of the HTML document.

<body>

Contains the content of the HTML document.

Heading tag: <h1>, <h2>, ... <h6>

Define headings of different levels.

Paragraph tag: <p>

Defines a paragraph.

Anchor tag: <a>

Defines a hyperlink.

Image tag: <img>

Embeds an image.

List tags: <ul>, <ol>, <li>

Define unordered and ordered lists.

Div tag: <div>

Defines a division or a section in an HTML document.

Span tag: <span>

Defines a small piece of content within a larger element.

Break tag: <br>

Inserts a line break.

<strong> and <em> tag.

These tags are used for text emphasis. < strong> represents strong importance or importance, typically displayed as bold, while < em> represents emphasized text, typically displayed as italic.

This is strong and emphasized text.
